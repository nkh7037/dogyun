## 狗云有什么特别的地方？

说几个真实的亮点，不是说废话的那种。

**机龄计划**算是比较有意思的设计——你用的时间越长，每小时获得的免费流量额度就越高。比如香港-MG 国际线路，从 0 个月的每小时 50MB 一路涨到第 12 个月的每小时 650MB，换算下来相当于用越久越省钱，而且续费同价。这个设计对长期用户确实挺友好的，不存在"越续费越贵"的情况。

**弹性云的暂停功能**也挺实用，不用的时候可以暂停留机，只收硬盘和 IPv4 费用，CPU、内存、流量都不计费。适合那种"偶尔用一次，但不想每次都重新配置"的场景。

**自主研发的 Dudu 控制面板**支持在线重装系统、挂载 ISO、更换 IP、切换线路、调整配置，操作基本上不需要提工单，全程自助。

**线路选择**方面，香港 KC 机房支持精品线路（CN2+CU+CMI 三网优化），香港 MG 机房最大带宽可到 1Gbps 国际路线，适合需要跑国际流量的场景。
